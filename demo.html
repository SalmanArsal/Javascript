Types Of JAVASCRIPT

We can implement Js code into a webPage in two ways:
1.Internal javascript
2.External javascript

1.Internal javascript

1. we use script tag in head or body of the html document , (recomended in body) 

            Syntax:
                   <script>

                            // JS code...

                   </script>

2.External javascript

1.We use an External javascript file created with .js Extension
2.We need to link that external Js file in html document to implement js code into a webPage.
3.script tag is used to link external javascript file.
